Fr1zeen

Planned Release Date: November 16

What even is Fr1zeen?

Fr1zeen is a newly-discovered exploit that can unenroll an enterprise enrolled chromebook. This exploit uses RMA shims, which is only intended for use by manufacturers and repair shops, but were leaked. By using this exploit, we can boot into developer mode, enter recovery and boot off of the shim USB like it was a recovery image. Since Google left a rootfs vulnerability unpatched, we were able to run unsigned code; and therefore unenrollment.

Okay, so, how did you find Fr1zeen?

Even though Google has patched unenrollment, the new exploit was discovered by a group of ethical hackers during a routine security audit of school Chromebooks. The hackers identified a vulnerability in the update-engine that could be exploited to remove enterprise enrollment, leading to the creation of the exploit. The exploit uses RMA shims to manipulate update-engine to allow for unenrollment when rebooting back into ChromeOS.

What are the requirements for Fr1zeen?

  1. A brain
  2. An enterprise enrolled chromebook with a working leaked RMA shim
  3. Access to an image flasher like Chromebook Recovery Utility
  4. A USB flash drive with at least 8gb of storage
